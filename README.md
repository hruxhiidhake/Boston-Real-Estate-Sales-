# Boston-Real-Estate-Sales-

## Project Summary: Boston House Price Analysis

This project focuses on analyzing housing data in Boston’s competitive real estate market to uncover key factors influencing property prices. By studying trends in location, property features, and market conditions, the goal is to support data-driven decision-making.

As a Lead Data Analyst, the objective is to generate actionable insights that help the company:

## Business Objective

1. Improve pricing strategies
2. Optimize property recommendations
3. Better match customer preferences

Overall, the analysis aims to enhance the house recommendation system, enabling more accurate, personalized, and competitive offerings for buyers while improving business performance.
